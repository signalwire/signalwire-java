# PORT_SIGNATURE_OMISSIONS.md

Java signature divergences from the Python reference. Each entry documents a
shared symbol whose parameter list, return type, or both differ from Python's
canonical definition. The audit (`diff_port_signatures.py`) excuses any drift
listed here; un-listed drift fails CI.

Five-bucket classification (see PORTING_GUIDE.md):

  1. **Rename only** — handled by enumerate_signatures.py rename tables, not here.
  2. **Idiomatic addition** — handled by PORT_ADDITIONS.md, not here.
  3. **Signature divergence** — Builder pattern collapses N kwargs into 1
     typed wrapper; fluent self-return where Python is void; typed
     functional-interface where Python is a generic Callable; primitive
     boxing where Python uses a union; etc. **This file.**
  4. **Deliberate omission** — handled by PORT_OMISSIONS.md, not here.
  5. **Real gap** — must be fixed in source, never excused.
